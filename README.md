# 🚀 QuickBlog – AI-Enhanced Blogging Platform

## 📖 Overview

QuickBlog is a full-stack AI-powered blogging platform built using the MERN stack. It allows users to create, manage, and publish blogs with an intuitive editor, secure authentication, image uploads, and an admin dashboard.

---

## ✨ Features

- 🔐 JWT Authentication & Authorization
- 📝 Rich Text Blog Editor
- 📷 Image Upload Support
- 🤖 AI-Assisted Content Creation
- 📱 Responsive Design
- 📊 Admin Dashboard
- 📚 Blog Management System
- ☁️ Cloud-Based Image Storage
- 🚀 Fast and Scalable MERN Architecture

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Storage
- ImageKit

### Authentication
- JWT
