## 📖 Overview

QuickBlog is a full-stack AI-powered blogging platform designed to simplify content creation and management. Users can write, publish, edit, and manage blogs while leveraging AI features to improve content quality and productivity.

The platform offers secure authentication, rich text editing, responsive design, and AI-assisted blogging capabilities.

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/QuickBlog.git
cd QuickBlog
```

### 2️⃣ Install Dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

### 3️⃣ Configure Environment Variables

Create `.env` file inside server folder:

```env
PORT=5000

MONGODB_URI=your_mongodb_connection

JWT_SECRET=your_secret_key

OPENAI_API_KEY=your_api_key
```

### 4️⃣ Run Project

#### Backend

```bash
npm run server
```

#### Frontend

```bash
npm run dev
```
